System Changes
===

### Defense Changes

* Defense formula changed:
    * Old formula:
    <p><code>DMG * (2 - DEF)</code></p>

    * New formula:
    <p><code>SCALE = 0.5</code></p>
    <p><code>DMG / (DEF * SCALE + (1 - SCALE))</code></p>

### Weapon Changes

* Reload speed is now affected by Cooldown Modifier.
